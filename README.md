# Hexo setup for [1 Pixel Out](http://1pixelout.net)

## Install

First, install the [Hexo](https://hexo.io) CLI globally:

```
> npm install -g hexo
```

Clone this repo and install node modules:

```
> npm install
```

## To run the local server

```
> npm start
```

## To run the functional tests

```
> npm test
```

## To deploy

```
> npm run deploy
```
